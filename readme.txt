ถ้อยแถลงแห่งวิถีแห่ง Agile 
http://agilemanifesto.org/

และนี่คือวิถีแห่ง Agile Zen
หรือมันจะสุดขั่ว แต่ก็ตามนี้
คุณค่า: คุณค่าสูงสุดของเรา คือจักต้องส่งมอบซอฟแวร์ที่ แจ่มเจ๋งสุดๆให้ลูกค้าได้ทุกเวลาเมือต้องการ
วิถีแห่ง Agile Zen:
-> เราจะให้บริการลูกค้า หรือลูกค้าต้องมาก่อน เหนือกว่า แผนที่วางไว้อย่างเลิศหรูอลังการ
-> เราจะตอบสนองการเปลี่ยนแปลงได้แทบจะทันที เหนือกว่า การทำตามแผนที่เลิศหรูอลังการที่ต้องทำตาม
-> เราจะประดิษฐ์ซอฟแวร์ที่ทำงานได้อยู่ทุกเวลา เหนือกว่า เอกสารที่สมบูรณ์แบบสุดๆก่อนถึงจะประดิษฐ์ซอฟแวร์ได้
-> เราจะเค้นเอาความจริงจากลูกค้าให้ได้แบบตัวต่อตัว เหนือกว่า กระบวนการหรือเครื่องมือใดๆที่จะเข้ามาขัดการเค้นความจริงจากลูกค้า ของเราไปได้ไม่มี

อ่านแล้วมันค่อนข้างที่จะเวอร์ เป็นไปไม่ได้ แต่เราก็จะปฏิบัติตามวิถีแห่ง Agile Zen นี้ตลอดไป (ดื้อ)
